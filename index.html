Discover Snapsort, your sorting ally

Open-source desktop utility application for sorting, analyzing and organizing your files in folders sorted by year and month.

Download Visit GitHub

Features

  • Sorting

    Select a folder and your files will be copied and sorted into folders according to year and month.

  • Analyzing

    Scan a folder to make sure all files are correctly sorted in your folders.

  • File extensions

    Select the file extensions to be taken into account when sorting.

  • Custom months

    Personalize the names of your months according to your needs and desires.
